Технологии Роксофт в действии: мы запустили первый биометрический постамат в МФЦ Ленинградской области

1 апреля 2026

Совместно с Центром биометрических данных, компаниями Ovision и Evolenta, а также МФЦ Ленинградской области мы реализовали пилотный проект по выдаче документов через постамат с использованием биометрической идентификации.

Первый терминал начал работу в отделении «Бугры» филиала МФЦ «Мурино». Оборудование представляет собой постамат на 49 ячеек, оснащенный биометрическим модулем для распознавания лица. Это решение позволяет заявителям получать готовые документы без предъявления паспорта и ввода кодов из SMS-сообщений.

Процесс получения услуги максимально упрощен благодаря интеграции наших аппаратно-программных комплексов с Единой биометрической системой. Заявитель подходит к терминалу, смотрит в камеру биометрического сканера, и ячейка с документами открывается автоматически. Процедура занимает несколько секунд и не требует использования дополнительных носителей информации.

Внедрение биометрии значительно увеличило перечень услуг, доступных для получения через автоматизированный терминал. Если через традиционный SMS-код граждане могли забрать документы только по 75 услугам, то биометрическая идентификация открыла доступ к 291 услуге.

Традиционный способ получения документов по цифровому коду через чат-бот МФЦ также сохраняется, чтобы каждый заявитель мог выбрать удобный для себя вариант.

Для компании Rocksoft этот проект — еще один шаг в развитии экосистемы удобных государственных сервисов. Мы продолжаем работу над масштабированием решения и готовы внедрять подобные комплексы в других регионах России.
